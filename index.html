CloudConvert API 키 설정

CloudConvert API 키를 입력하세요. 키는 브라우저에만 저장되며 서버로 전송되지 않습니다.

  1. cloudconvert.com에서 무료 계정 생성
  2. Dashboard → API Keys → Create API Key 클릭
  3. 발급된 키를 아래에 입력 (무료: 하루 25회 변환)

왜 우리 PDF 변환기를 사용하나요?

HWP 완벽 지원

한글 문서(.hwp, .hwpx)를 PDF로 완벽하게 변환합니다. 레이아웃과 폰트가 그대로 유지됩니다.

고품질 변환

원본 문서의 레이아웃, 이미지, 폰트를 완벽하게 보존하여 고품질 PDF를 생성합니다.

빠른 변환 속도

클라우드 기반 변환 엔진을 사용하여 대용량 파일도 빠르게 변환됩니다.

보안 처리

업로드된 파일은 변환 즉시 삭제됩니다. 개인정보와 문서 내용은 안전하게 보호됩니다.
